# /hn - Hacker News 日本語ダイジェスト

Hacker Newsのトップ記事を取得して、日本語で初学者にもわかりやすく要約してください。

## 手順

1. まず `python3 ~/hn-digest/scripts/fetch_hn.py` を実行して直近24時間以内のトップ記事を取得
2. 取得結果からスコア＋コメント数の合計値で上位10〜15件を選定（求人系は除外）
3. すべての記事を同じ詳しさで要約する（スコアの高低で差をつけない）
4. 各記事のURLにアクセスして内容を読む（タイトルだけで推測しない）
5. HNのコメントページにもアクセスし、目立った意見を複数引用する
6. 以下のフォーマットで各記事を日本語要約:

```
## [番号]. 記事タイトル（日本語訳）
**原題**: 英語の元タイトル
**URL**: 記事のURL
**HN**: HNディスカッションURL
**スコア**: ○○ / **コメント**: ○○件 / **合計**: ○○

**ざっくり言うと:**
（2〜3文。背景知識がなくてもわかるように）

**もう少し詳しく:**
（3〜5文。専門用語には簡単な注釈）

**なぜ注目？:**
（なぜ話題なのか、業界への影響、学習者向けポイント）

**HNコメントで目立った意見:**
- 「要約」（ユーザー名）— 補足
```

7. 最後に用語集（5〜10個）を追加
8. `~/hn-digest/docs/digests/YYYY-MM-DD.md` として保存（YYYYMMDDは今日の日付）。ファイルの先頭に以下のfront matterを必ず付けること:
```
---
layout: digest
title: "HN Digest — YYYY-MM-DD"
---
```

## 翻訳ルール
- 自然な日本語で。直訳は避ける
- 技術用語はカタカナ + 簡単な説明
- 「で、結局これ何が重要なの？」にちゃんと答える
- カジュアルなトーンでOK
- 用語の注釈は「単語（説明）」の形式にする。単語自体を（）で囲まないこと
  - ○ GPT（Generative Pre-trained Transformer: ChatGPTなどの基盤となっているAIモデルの設計思想）
  - × （GPT（Generative Pre-trained Transformer）: ChatGPTなどの基盤となっているAIモデルの設計思想）
